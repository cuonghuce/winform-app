1. Download thư mục từ gg drive về thì tiến hành giải nén 
2. Mở Microsoft SQL Server Management Studio, kéo file script.sql vào và click execute.
    + ShopSimpleClassicDemo.sql có chứa dữ liệu.

    + ShopSimpleClassicNoData.sql không chứa dữ liệu.

3. Sau đó tìm file ShopSimple.sln trong thư mục ShopSimple và mở bằng Visual Studio IDE (Visual Studio 2022 hoặc Visual Studio 2019, ...).

4. Tìm Settings.settings trong view Solution Explorer (ShopSimple (tên project) > Properties > Settings.settings) và double click vào và thay đổi dữ liệu ở cột Value hoặc Click phải vào ShopSimple (Tên project/ dự án) > Properties > Settings và Đổi giá trị kết nối ở cột Value
